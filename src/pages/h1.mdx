# h1

## Syntax

```
# h1 Heading
```

## Examples

Short heading that does not wrap:

# Short Heading

Long heading that wraps:

# Adipisicing do eiusmod adipisicing nisi commodo minim sit sit dolore velit. Velit esse velit dolore magna consectetur est sint qui ad duis est pariatur ex. Cillum nulla labore ea cillum. Ullamco eu proident minim Lorem dolor elit aute magna dolore eu commodo.
